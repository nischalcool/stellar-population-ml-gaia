DATA INSTRUCTIONS
=================

Raw data is NOT included in this repository due to file size.

All data was queried directly from the ESA Gaia DR3 archive
using astroquery. To reproduce the dataset:

1. Run notebook 01_data_query.ipynb
2. This will query gaiadr3.gaia_source for all four clusters
3. Cleaned data will be saved as CSV files automatically

Gaia Archive: https://gea.esac.esa.int/archive/
Gaia DR3 Paper DOI: 10.1051/0004-6361/202243940
